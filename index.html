Computer Basics

Don't know much about computers? Don't worry.

You don't need any experience to join. Before we get into AI, we make sure everyone is comfortable with the tools. We start from zero, no judgment.

>_

Terminal

A text window that controls your whole computer. Sounds intimidating. It's not. You'll be running commands confidently within the first session.

⌂

Finder and File Explorer

Where your files actually live. We'll teach you how to organize your projects, stop losing things in Downloads, and navigate your computer like you own it.

/

File Paths

What does /Users/you/Documents actually mean? Once it clicks, you'll wonder how you ever got by without understanding it. Takes about five minutes.

⌘

Keyboard Shortcuts

The ones everyone knows, plus the ones that actually save time. Copy, paste, undo, switch apps, take screenshots. Small habits, big difference.

{ }

Text Editors

VS Code is where you write code. We'll get you set up, walk you through the layout, and make sure nothing about it feels foreign before we start building.

< >

Browser DevTools

Right-click any website and inspect it. You can see exactly how it's built. It's one of the most useful things you can learn, and most people have no idea it exists.
